# 🚀 Python Power Up

Automação de cadastro de produtos utilizando **Python**, **PyAutoGUI** e **Pandas**.

Este projeto realiza a leitura de uma planilha Excel contendo informações de produtos e automatiza o preenchimento de um formulário web, eliminando tarefas repetitivas e aumentando a produtividade.

---

## 📌 Funcionalidades

* Leitura automática de uma planilha Excel;
* Cadastro de produtos em um sistema web;
* Preenchimento automático dos campos do formulário;
* Simulação de teclado e mouse com PyAutoGUI;
* Automatização completa do processo de cadastro.

---

## 🛠️ Tecnologias Utilizadas

* Python 3
* Pandas
* PyAutoGUI
* OpenPyXL
* Time

---

## 📚 Aprendizados

Este projeto foi desenvolvido para praticar:

* Automação de processos com Python;
* Manipulação de arquivos utilizando Pandas;
* Automação de interface gráfica com PyAutoGUI;
* Organização de projetos em Python.
